Java SWING desktop application



A desktop GUI version of this site is available for download. It can run on either WINDOWS or LINUX. It has not been tested on Apple MAC.


It is a Java SWING application that was built with Java 18. It is available as a single JAR file (7.8 Mb). A Java run-time (30 Mb) is also available to run the application on WINDOWS, if you do not have Java already installed on your PC.


The PostgreSQL (v14) database for this application is also available as a BACKUP/SQL files. It is about 40Mb in size and was created in pgAdmin4.


Each of the above files is available here
